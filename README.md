# puzzle solver with ZDD

Note: Almost all of the code in this project is written by AI (Claude, Sonnet 4.6). 

## puzzle types

- peg solitaire : /zdd_puzzle/myproject/solit
- diamond (personal) : /zdd_puzzle/myproject/diamond

## usage
This project uses KyotoDD (SAPPOROBDD 2.0).

See https://github.com/graphillion/kyotodd for API and https://graphillion.org/kyotodd/ for doc.

## example
```
$ cd zdd_puzzle/myproject/solit
$ python3 solit.py
```

Then, an HTML file will be created.
